About Halloween

ハロウィンとは、万聖節（ばんせいせつ）（11月1日）の前夜祭のこと。
古代ケルト起源で，秋の収穫を祝い悪霊を追い出すための祭りです。
カボチャをくりぬき目鼻口をつけた『ジャックオランタン』という提灯を飾り，
お化けの仮装をした子供たちが 「Trick or treat!(ごちそうくれなきゃイタズラするぞ)」
と言いながら近所を回り、お菓子もらうのが恒例となっています。
